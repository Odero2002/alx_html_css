# ALX CSS Advanced Project ## Objectives This project builds on the HTML Advanced project by focusing on styling a complex webpage using CSS. The design is based on a Figma file provided by ALX, and all styling must be implemented using pure CSS—no external libraries or frameworks. ## Features - Semantic HTML structure following best practices - Responsive design for desktop and mobile - Custom fonts and images as specified in the Figma file - Sections include: Header, Banner, Quote, Videos, Membership, FAQ, and Footer ## Getting Started 1. Clone this repository. 2. Open `css_advanced/index.html` in your browser. 3.
